# 📱 Réplica de la Red Social X

## 📖 Descripción

Este proyecto consiste en la creación de una página web inspirada en la interfaz de la red social **X (anteriormente Twitter)** utilizando únicamente **HTML5** y **CSS3**.

El objetivo principal fue aplicar conceptos fundamentales de desarrollo web frontend, especialmente el uso de **Flexbox** para la distribución y alineación de elementos, así como implementar aspectos básicos de **diseño responsivo** para mejorar la visualización en diferentes tamaños de pantalla.

---

## 🚀 Tecnologías Utilizadas

- HTML5
- CSS3

---

## 🛠️ Conceptos Aplicados

### Flexbox
Se utilizaron diversas propiedades de Flexbox para construir la estructura de la página:

- `display: flex`
- `flex-direction`
- `align-items`
- `justify-content`
- `flex-wrap`
- `gap`

### Diseño Responsivo
Se realizaron ajustes básicos para adaptar la interfaz a distintos tamaños de pantalla mediante:

- Media Queries
- Unidades relativas
- Distribución flexible de elementos

---

## ✨ Características

- Barra lateral de navegación.
- Sección principal de publicaciones.
- Panel lateral con contenido adicional.
- Diseño inspirado en la interfaz de X.
- Distribución de elementos mediante Flexbox.
- Adaptación básica para diferentes dispositivos.

---

## 🎯 Objetivos de Aprendizaje

Este proyecto permitió reforzar conocimientos relacionados con:

- Estructuración de páginas web con HTML.
- Aplicación de estilos mediante CSS.
- Uso práctico de Flexbox para la maquetación.
- Diseño responsivo.
- Organización y buenas prácticas en el desarrollo frontend.
